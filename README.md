# Centro Comercial TEI — proyecto

Este repositorio contiene el backend y frontend del proyecto "Centro Comercial TEI".

Sección relevante: Backend
- `backend/README.md` — instrucciones rápidas en inglés para instalar Alembic y ejecutar migraciones/pruebas.
- `backend/README-ES.md` — versión en español del README del backend.
- `backend/setup/Install-Alembic.ps1` — script PowerShell que crea un virtualenv e instala dependencias (incluye Alembic).
- `scripts/run_postgres_and_tests.ps1` — script para levantar Postgres en Docker, aplicar migraciones y ejecutar la prueba de concurrencia.

Siga las instrucciones dentro de `backend/README.md` o `backend/README-ES.md` para configurar el entorno y ejecutar migraciones.
